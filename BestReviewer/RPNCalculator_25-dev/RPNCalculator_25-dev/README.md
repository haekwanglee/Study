### 후위 연산 계산기  
- “1 2 +”           =>  1 + 2 = 3  
- “1 2 –”           =>  1 - 2 = -1  
- “2 3 *”           =>   2 * 3 = 6  
- “20 5 /”          => 20 / 5 = 4  
- “20 0 /”          => Divided by Zero Exception  
- “4 2 + 3 *”       => (4 + 2) * 3 = 18  
- “3 5 8 * 7 + *”   => ((5 * 8) + 7) * 3 = 141  
- “9 SQRT”          => √9 = 3     Math.sqrt(9) 사용  
- “4 5 MAX 1 2 MAX *”   => MAX(4, 5) * MAX(1, 2) = 10  
- “5 3 9 2 4 1 MAX”     => MAX(5, 3, 9, 2, 4, 1) = 9  
  
  
 *** 모든 수는 정수이며, 나누기 연산을 포함하여 결과는 정수인 경우만 입력됩니다.
